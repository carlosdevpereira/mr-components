# Dropdown

The dropdown component is a simple implementation of a Button component, that triggers a dropdown menu when clicked. The content of the dropdown menu can be fully costumized through a default slot.

  • API
    • Props
  • Basic Dropdown
  • Button
    • Themes
    • Sizes

# API

# Props

NameTypeDefaultDescription
...ButtonAll the props from the Button component are also available here and are applied directly to the Button instance

# Basic Dropdown

<template>
	<Dropdown label="Click me">
		<Button theme="text" class="mr-dropdown-option">Option 1</Button>
		<Button theme="text" class="mr-dropdown-option">Option 2</Button>
		<Button theme="text" class="mr-dropdown-option">Option 3</Button>
	</Dropdown>
</template>

# Button

By default, the dropdown accepts all the props also accepted by the Button component (themes, variants, sizes, etc). All those props are passed down automatically to the button instance.
